# DAL Reproducibility Initiatives
GitHub for Reproducible Experiments

---

## Experiment Index
- [cca/](cca/) - Contextual-Compositional Associations for neonatal mortality rate monitoring; scripts and notebooks for NMR analysis.
- [chiron/](chiron/) - Chiron: parallel engine for algebraic scientific workflows; pointers to paper, site, and demo.
- [DAI2M/](DAI2M/) - Data-Centric AI for ethanol production forecasting; ARIMA and PRE+MLM pipelines with rolling-origin evaluation.
- [fedd/](fedd/) - Fuzzy Ensemble for Drifter Detection (3W oil wells); scripts for detectors, combinations, and ensemble analysis.
- [flight-delay-association-rules/](flight-delay-association-rules/) - Association rules for Brazilian domestic flight delays; study guide and semantic workflow plan.
- [ftdctsad/](ftdctsad/) - Fine-Tuning Detection Criteria for Time Series Anomaly Detection; benchmark script and results.
- [mgc/](mgc/) - Mixed Graph Complementarity; pointers to MGF repo and suggested semantic organization.
- [msed/](msed/) - Multi-Scale Event Detection in financial time series; multiscale pipeline and Harbinger examples.
- [rt/](rt/) - Resilient Transformation for anomaly detection in heteroscedastic time series; RT/RTAD scripts and metrics.
- [softed/](softed/) - SoftED: soft evaluation metrics for time series event detection; datasets, metrics, and experiment scripts.
- [sst/](sst/) - Sea Surface Temperature prediction with temporal aggregation experiments.
- [stmotif/](stmotif/) - Spatio-Temporal Motifs Discovery (CSA) examples and figures.
- [stmotifexplorer/](stmotifexplorer/) - STMotif Explorer tool overview and assets.
- [tsfm/](tsfm/) - LLMs for low-frequency time series forecasting (Chronos, Lag-LLaMA) vs baselines.
- [tspredit/](tspredit/) - TSPredIT: integrated tuning of preprocessing and forecasting; workflows and analysis.

Each folder contains its own `readme.md` with objectives, structure, and how to run its experiments. External documentation links (when applicable) are placed inside the corresponding folder README.
